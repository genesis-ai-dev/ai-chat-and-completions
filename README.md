# MINI AI COPILOT: A Smart Programming Assistant

## Features

- 支持任何兼容openai接口的本地LLM。
- 完全支持中文交互。
- 方便进行代码补全。
- 提供AI聊天进行代码相关问答。
- 支持选定代码片段的问答。

## Current Limitations

- 单个输入的最大长度不能超过4000个字符。

## User Guide

- 代码补全：按alt+q快捷键，或从右键菜单中选择“自动补全”。之后，按“tab”进行输入。
- AI聊天：从侧边菜单中访问此功能。您可以选择代码并对其进行查询。
